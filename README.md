Информационная панель системы кв152 автономная по wifi, данные берет из mqtt  реализована на основе ESP8266

  Подключение NodeMCU к LCD1604 через i2c

   NodeMCU I2C 1602 LCD
   - Vin VCC<BR>
   - GND GND
   - D1  SCL
   - D2  SDA

   Подключение датчика движения
   - Vin VCC
   - GND GND
   - D3  OUT
